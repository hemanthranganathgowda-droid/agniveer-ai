# AgriVerify AI

AI-powered crop damage verification platform for farmers.

## Features
- GPS verification within 30 meters
- AI crop image validation
- Fake/screenshot detection
- Automatic QR approval
- Government scheme recommendations
- Real-time dashboard
  ## Live Demo
  https://agri-swift-verify.lovable.app

## Tech Stack
- Lovable
- React
- TypeScript
- Vite

## Project Goal
To reduce paperwork and speed up crop damage verification using AI and GPS technology.
